# Diwali Sales Analysis

## 📊 Project Overview

This project performs a comprehensive analysis of sales data during the Diwali season. It leverages Python data analysis and visualization techniques to uncover patterns, trends, and insights about customer purchasing behavior during India's most significant shopping festival.

## 🎯 Objectives

- Analyze sales trends during the Diwali season
- Identify key customer demographics and segments
- Understand purchasing patterns across different product categories
- Provide actionable insights for business decision-making and targeted marketing strategies
- Visualize data patterns to support data-driven conclusions

## 📈 Key Findings

**Primary Insight:**
- Unmarried women aged 26-35 years from Uttar Pradesh, Maharashtra, and Karnataka working in IT, Healthcare, and Aviation sectors are the most likely customers to purchase from Food, Clothing, and Electronics categories.

## 💾 Dataset

The project analyzes Diwali sales data including:
- Customer demographics (age, gender, marital status)
- Geographic information (states/regions)
- Occupational details
- Product categories and purchase information
- Sales transaction records

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3.7 or higher installed on your system.

### Installation

1. **Clone the repository:**
```bash
git clone <repository-url>
cd Diwali_Sales_Analysis
```

2. **Install required packages:**
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

Or using requirements.txt:
```bash
pip install -r requirements.txt
```

3. **Launch Jupyter Notebook:**
```bash
jupyter notebook
```

4. **Open the notebook:**
   - Navigate to `Diwali_Sales_Analysis_Python_Project.ipynb`
   - Run the cells sequentially to execute the analysis
